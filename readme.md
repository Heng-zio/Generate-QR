# QR Code Generator

A simple Node.js application that generates a QR code from a user-provided URL.

## Features

* Enter a URL through the command line.
* Generates a QR code image (`qr_img.png`).
* Saves the entered URL to `URL.txt`.

## Installation

```bash
npm install
```

## Usage

```bash
node index.js
```

Enter a URL when prompted, and the application will generate the QR code and save it automatically.
